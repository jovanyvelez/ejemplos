Bienvenido a la Aplicación Web CRUD

Esta es una aplicación web de demostración que muestra cómo crear operaciones CRUD (Crear, Leer, Actualizar, Eliminar) usando:

  • FastAPI - Framework web moderno y rápido para Python
  • SQLAlchemy - ORM para manejo de base de datos
  • Jinja2 - Motor de plantillas para renderizar HTML
  • SQLite - Base de datos ligera para almacenamiento
